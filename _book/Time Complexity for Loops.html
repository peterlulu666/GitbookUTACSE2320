Conventions

1.The time complexity of this code would be 5 + 5 + 5 = 3×53 \times 53×5 = 15

#include <stdio.h>

int main() {
    for (int i = 0; i < 3; ++i) {
        printf("\nA: ");
        for (int j = 0; j < 5; ++j) {
            printf("B");
        }
    }

    return 0;
}

The output would be

A: BBBB
A: BBBB
A: BBBB

The time complexity of this code would be

\[ \underbrace{N+⋯+N}_{M\text{ times}} = M \times N \]

for (int i = 0; i < M; ++i) {
        printf("\nA: ");
        for (int j = 0; j < N; ++j) {
            printf("B");
        }
    }

The output would be

A: BBBB...BBBB
A: BBBB...BBBB
..............
A: BBBB...BBBB

  • We would use Θ\ThetaΘ to indicate the time complexity of the code.
